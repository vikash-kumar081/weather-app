🌦 About Weatherly

Weatherly is a modern weather forecasting web application designed to provide real-time weather updates, temperature trends, humidity, wind speed, visibility, air quality insights, and multi-day forecasts through an intuitive and responsive interface.

This project was developed by Vikash Kumar, a B.Tech Computer Science student, using HTML, CSS, JavaScript, Chart.js, and Weather APIs. The application demonstrates skills in frontend development, API integration, responsive web design, data visualization, and modern UI/UX principles.

The goal of this project was to create a clean, interactive, and user-friendly weather dashboard that works seamlessly across desktop, tablet, and mobile devices while delivering accurate weather information in real time.
