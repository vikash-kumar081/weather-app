# 🌦️ Advanced Weather Forecast Web App

A modern weather forecasting web application built using HTML, CSS, and JavaScript.

## Features

- Real-time Weather Updates
- Voice Search
- Air Quality Index (AQI)
- Sunrise & Sunset Timings
- Modern Glassmorphism UI

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
- GitHub Pages
